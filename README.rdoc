= Project priorities plugin

Плагин добавляет в Redmine возможность использовать приоритеты задач, в области видимости проекта.
По-умолчанию, приоритеты являются общесистемной настройкой, однако по ряду причин - это неудобно.
Плагин добавляет новую вкладку в "настройки проекта" - "Приоритеты".
В ней содержится список системных приоритетов, можно выбрать те, которые нужны отностительно текущего проекта.
Остальные приоритеты будут игнорироваться в рамках текущего проекта (кроме использования фильтра)
